SP-Chatbot is a web-based project developed as part of my final year diploma. This website provides detailed information about planets in a simple and interactive way. Users can explore information about each planet, including basic characteristics, orbital data, and other scientific facts.
The project focuses on front-end web development using HTML, CSS, and JavaScript. Although it is not hosted yet, all source code files are included in this repository.Features
Planet Information: Users can view information about all planets in the Solar System.
Interactive Design: Easy-to-navigate pages with clear organization for each planet.
Responsive Layout: Works on desktop and mobile devices.
Educational Purpose: Ideal for students and anyone interested in astronomy.
Future Ready: Can be easily enhanced with animations, API integration, or hosting online.Technologies Used
HTML5 – Structure of the website
CSS3 – Styling and responsive design
JavaScript – Interactivity and dynamic behavior
